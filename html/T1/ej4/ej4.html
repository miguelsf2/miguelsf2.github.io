Ejercicio 4: Bubble Shooter

Enlace para ver el codigo

En este ejercicio se pide hacer un buble-shooter sencillo con velocidad constante.

Para realizarlo, he usado los siguientes parámetros:

  • Tamaño de ventana:(600x600)
  • Radio pelotas: 50 px
  • Velocidad pelota: 200 px
  • Posición iniciar: (WIDTH/2, HEIGHT/2)
  • N_X = WITH / R
  • Filas = 3

La variable N_X indica el número de pelotas rojas que se generaran horizontalmente en la parte de arriba (en el caso de la imagen, 12) y la variable filas indica el número de filas de bolas rojas que se generarán.
